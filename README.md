# Caffeinate Timer

Mac の自動スリープを一時的に防ぐための、シンプルで柔軟なターミナルベースのタイマーツールです。

プレゼンテーション中、大きなファイルのダウンロード中、あるいは動画の書き出し中など、「今だけは Mac にスリープしてほしくない」という時に、指定した時間だけスリープを防止します。
macOS 標準の caffeinate コマンドを、より直感的かつ簡単に使えるようにラップしています。

⸻

Features
	•	柔軟な時間指定
「分」だけの入力、コロン（:）区切り、h/m/s の単位表記、さらには小数（例: 1.5h）など、人間が直感的に思いつく様々なフォーマットに対応しています。
	•	全角・表記ゆれ対応
うっかり全角で「１ｈ３０ｍ」と入力してしまったり、「1 hour」のようにスペースや長い単位をつけてしまっても、自動で整形して認識します。
	•	わかりやすい表示
スリープ防止の終了予定時刻と継続時間をターミナル上にカラー表示します。
	•	簡単起動
.command ファイルなので、ターミナルを開いてコマンドを打ち込む必要はなく、Finder からダブルクリックするだけで起動できます。

⸻

Requirements

Environment	Details
macOS	標準搭載の caffeinate コマンドと bash を使用します。


⸻

Usage
	1.	このリポジトリをダウンロード（またはクローン）します。
	2.	初回のみ、ファイルに実行権限を付与する必要がある場合があります。ターミナルを開き、以下のコマンドを実行してください。
        chmod +x /path/to/caffeinate.command
	3.	Finder から caffeinate.command をダブルクリックして起動します。
	4.	ターミナルの画面が開き、時間の入力を求められるので、希望する時間を入力して Enter を押します。
	5.	タイマーが完了するか、Ctrl + C で中断するまで、Mac のスリープが防止されます。

⸻

Input Format Examples

非常に多様な入力方法に対応しています。

入力例	解釈される時間	備考
90	90分	数字のみの場合は「分」として扱われます
1.5	1分30秒（90秒）	小数での「分」指定
1:30	1分30秒	分:秒
1:30:00	1時間30分0秒	時:分:秒
1h または 1hour	1時間	単位付き（時間）
45m または 45min	45分	単位付き（分）
20s または 20sec	20秒	単位付き（秒）
1h30m20s	1時間30分20秒	複合指定
1.5h	1時間30分	小数での「時間」指定

Note: 全角数字（例: ９０）や、全角アルファベット（例: １ｈ）で入力しても正常に動作します。

⸻

Stop / Cancel
	•	指定した時間が経過すると、自動的にスリープ防止が解除され、ターミナル上に完了メッセージが表示されます。
	•	途中でスリープ防止を解除したい場合は、ターミナルウィンドウ上で Ctrl + C を押してください。安全に中断処理が行われます。
